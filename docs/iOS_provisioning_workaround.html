iOS Provisioning Workaround

What if you have no provisioning profile?

  1. Disable push notifications for the project.

  2. Select StatusIm in the Project Navigator.

  1. Select StatusIm from the list of targets.

  2. Set the signing to "automatic".

  3. Rename project name with your prefix.

  4. Rename unit-tests name with your prefix.
